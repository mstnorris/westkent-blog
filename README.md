# Simple Blog

This is a simple blog for West Kent students using PHP, PDO, & MySQL.

![Screenshot](http://i.imgur.com/z14xjVo.png)

## Features

- PHP
- PDO
- MySQL
- Bootstrap 4

## Installation

1. Install PHP
2. Install MySQL
3. Set up database
4. Create a user and grant them privileges
